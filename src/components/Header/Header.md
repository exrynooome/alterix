# Как добавить новые страницы?

Управление идет через `` const NavItems ``. Каждый NavItem содержит:

 - ``id`` (опционально) - нужен, если вы переходите на одной странице между секциями. Те же ``id`` нужно использовать далее на самой странице
 - ``label`` - текст, отображаемый в хедере
 - ``href`` - для роутинга между страницами

Вы можете использовать ``id`` и ``href`` одновременно
